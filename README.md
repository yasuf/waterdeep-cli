# Waterdeep CLI

## Installation

On Mac/Linux run

`$ npm install -g waterdeep-cli`

## Usage

### Retrieve a file on waterdeep by "username/filename" and send it to stdout

`$ water get [username]/[filename]`

`$ water get yasu/.gitconfig`

### Retrieves a list of files for a user

`$ water get [username]`

`$ water get yasu`
